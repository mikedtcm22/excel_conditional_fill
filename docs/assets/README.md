# FillGaps Icons

This directory contains icon assets for the FillGaps add-in.

## Required Icon Sizes

The manifest requires the following icon sizes:
- icon-16.png (16x16 pixels)
- icon-32.png (32x32 pixels)
- icon-64.png (64x64 pixels)
- icon-80.png (80x80 pixels)

## Placeholder Icons

For development and testing, placeholder icons are acceptable. The icon.svg file can be converted to the required PNG sizes using image processing tools.

## Production Icons

For production release, icons should:
- Use the FillGaps branding
- Be clear and recognizable at all sizes
- Follow Microsoft Office add-in icon guidelines
- Be in PNG format with transparent backgrounds
